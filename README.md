# 🎧 프로젝트 소개  

## 친구들과 약속 잡고 채팅하고 위치도 공유하는 앱  
### 1. 제목, 날짜, 장소, 맴버를 정하여 약속을 잡을 수 있습니다  
### 2. 캘랜더에서 나의 약속들을 한범넹 확인 할 수 있습니다  
### 3. 친구들과 채팅하며 약속 계획과 소통할 수 있습니다  
### 4. 약속 당일 친구들의 위치를 보며 어디까지 왔는지 확인 가능합니다  
<br>
